# 📚 StudyBuddy – Smart Study Planner
StudyBuddy is a browser-based study planner that helps students manage tasks, build daily study habits, and stay motivated. It features streak tracking, smart suggestions, XP progress, and badges, all in a simple, colorful interface that works offline with no login required.

StudyBuddy simplifies student productivity by combining task management with habit-building and motivation systems:

- Organizes tasks with subjects, priorities, and deadlines
- Encourages daily study through streak tracking
- Uses smart hints to suggest what to study next
- Keeps students engaged with rewards like XP, badges.

## 🎮 How to Use

### ➕ Adding a Task
1. Go to the **Home** tab
2. Type your task name (e.g. "Finish Maths homework")
3. Tap a **subject chip** (Maths, Science, etc.)
4. Tap a **priority button** (Urgent, High, Normal, Easy)
5. Pick a **due date**
6. Press **🚀 Add Task!**

### ✅ Completing a Task
- Tap the **circle button** on the left of any task to mark it done
- 🎊 Confetti will appear!

### 🔥 Logging a Study Day
- Go to the **Streak** tab
- Press **"I Studied Today!"** button once per day
- Watch your streak grow and earn badges!

### 📊 Viewing Progress
- Go to the **Progress** tab
- See your completion ring, subject breakdown, and priority split
